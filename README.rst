==================
Ansible Lint Rules
==================


Installation
============
Checkout this repository into a directory on your system or add it as a git
submodule to your project.


Usage
=====
Use ansible-lint with the options ``-R`` and ``-r <directory>``. E.g.

.. code-block:: Bash

  ansible-lint -R -r ./ansible-lint-rules site.yml
